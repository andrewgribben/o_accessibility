O Captain! My Captain! — Accessibility Demo

Progressive enhancement of Walt Whitman's poem (1865, public domain)

Five Stages of Accessibility

This demo shows how to progressively enhance a poem from basic HTML to a fully accessible experience. Each stage builds upon the previous one, demonstrating key accessibility principles.

Demo Stages

  1. Stage 1: Basic Text

    Simple paragraphs with line breaks. A semantic starting point.

  2. Stage 2: Visual Indentation

    Individual lines with CSS-based indentation to show poetic structure.

  3. Stage 3: Responsive Stripes

    Added striped backgrounds for small screens to help distinguish wrapped lines.

  4. Stage 4: Line Numbers

    Added line numbers.

  5. Stage 5: Full Accessibility

    Complete accessible implementation with ARIA, semantic HTML, and keyboard navigation.

About This Demo

This demonstration shows how to make poetry accessible while preserving its original structure and meaning. Each stage introduces new accessibility features without breaking the previous functionality.

Key Accessibility Features Demonstrated:

  • Semantic HTML structure
  • ARIA roles and labels
  • Keyboard navigation
  • Screen reader compatibility
  • Progressive enhancement
  • Responsive design
